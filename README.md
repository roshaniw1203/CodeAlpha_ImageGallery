# Image Gallery

A simple and responsive Image Gallery web application that displays a collection of images in an organized and visually appealing layout. This project demonstrates modern web design principles, responsive layouts, and interactive image presentation using HTML, CSS, and JavaScript.

## Features

* 🖼️ Responsive image gallery layout
* 📱 Mobile-friendly design
* ⚡ Fast and lightweight
* 🎨 Clean and modern user interface
* 🔍 Easy image browsing
* 💻 Cross-browser compatibility

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Use

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/image-gallery.git
   ```

2. Open the project folder.

3. Run the `index.html` file in your browser.
